# FinModel Builder

Веб-приложение для быстрого расчета финансовой модели бизнеса, стартапа или проекта под субсидии, гранты и меры поддержки.

## Ссылка на проект

https://kosulnikovworkspace-pixel.github.io/finmodel-builder/

## Возможности

- расчет прибыли в месяц
- расчет суммарной прибыли
- расчет срока окупаемости
- статус модели проекта
- предупреждения о рисках:
  - убыточность
  - низкая маржинальность
  - отсутствие ФОТ
  - неокупаемость за период
- сохранение данных через localStorage

## Технологии

- HTML
- CSS
- JavaScript
- Git
- GitHub Pages

## Цель проекта

Создать полезный MVP-инструмент для бизнеса и команд, которые готовят финансовые модели для получения мер поддержки.

## Планы развития

- сценарии модели:
  - оптимистичный
  - базовый
  - пессимистичный
- улучшенный UX
- форматирование чисел
